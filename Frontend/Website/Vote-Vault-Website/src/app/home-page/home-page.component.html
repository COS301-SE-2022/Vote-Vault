What is Vote Vault?

This voting system automates a voting process for your needs, while also making sure the votes are valid and secure

Ethereum blockchain powered smart contracts are used to save and verify user votes

In order to vote, users are verified by scanning their ID in the app

Instead of hard coding your forms for elections, they will be dynamically generated according to input provided by the voting official

View the various dashboards on this site that provide additional information about the voting process
